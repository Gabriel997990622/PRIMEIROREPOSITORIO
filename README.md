# Como criar um repositorio no GitHub
É necessario criar um repositoria no computador
vamos usar tambem o GitHub desktop

## Passo a passo 
-Suponde que você já tenha um projeto criado 
-Abrir o CMD Dentro da pasta e Digitar o comando. ```git init```
-Ir no Desktop e escolher a opção ```Add existing repository e selecionar a pasta projeto```
-Por fim fazer o comit e publicar. **Atenção**: lembrar de deixar sempre publico
